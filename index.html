Welcome to the Project Dev Rush! Please select your preferred project track based on your interests. The projects are designed to enhance your skills through a hands-on learning experience. Choose wisely, and let the journey of upskilling begin!

We will focus on two exciting projects during this program:

AI Tutoring System

Machine Learning Track

This project involves creating an intelligent tutoring platform using machine learning.

Key features include:

  • Personalizing learning paths for users
  • Forming topic-based study groups for collaboration
  • Providing instant feedback and adjusting teaching based on user sentiment
  • Generating personalized question banks for targeted practice

Book/Movie Recommendation System

Web Development Track

This project focuses on developing a recommendation platform while mastering CRUD operations.

Key features include:

  • Searching, adding, updating, and deleting book or movie entries
  • Generating tailored recommendations for users
  • Gaining practical experience in fundamental web development concepts

Choose the track that aligns with your interests and goals!

Here's how you can choose your project:

  1. Select a track (Web Development or Machine Learning)
  2. Install and set up the required tools and dependencies
  3. Follow along with the sessions
  4. Get hands-on experience while building your project

While it's not recommended due to the intensive nature of each track, you can attend and follow both tracks. However, we suggest focusing on one track to get the most out of the program and create a high-quality project.

You have flexibility in your tech stack choice:

  • Follow along with the taught stack during sessions
  • After January 20th, you can either:
    • Continue with the taught tech stack
    • Switch to your preferred tech stack for project completion
